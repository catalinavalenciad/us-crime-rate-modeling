US Crime Rate Modeling: Regression, Regularization, and Tree-Based Methods

This project demonstrates the design, evaluation, and comparison of regression and machine learning models to predict US crime rates using socio-economic indicators. The focus is on proper model validation, interpretability, and generalization performance rather than maximizing in-sample fit.

Business Context

Predicting city-level crime rates supports public policy, resource allocation, and socio-economic planning. The goal is to develop models that balance predictive accuracy, interpretability, and robustness under limited data.

Dataset

  • US Crime Dataset (UCI Machine Learning Repository)
  • Dataset
  • 47 cities, 15 numeric socio-economic predictors
  • 1 numeric response variable (Crime)

Methods

  • Data overview and outlier assessment (Grubbsâ€™ test)
  • Baseline linear regression (full and reduced models)
  • Principal Component Regression (PCR) for multicollinearity
  • Regularization: Stepwise, Ridge, Lasso, Elastic Net
  • Tree-based models: Regression trees and Random Forests

Key Results

  • Reduced linear regression generalized better than full model
  • Regularization improved stability but not predictive accuracy
  • Random forests reduced variance, slightly improving RMSE
  • Simple, interpretable models often outperform complex ones on small datasets

Limitations & Future Work

  • Small sample size (n=47) limits model complexity
  • Results sensitive to data splits and hyperparameters
  • Focus on prediction; causal inference not addressed
  • Future work: larger datasets, Bayesian uncertainty quantification, external data integration
